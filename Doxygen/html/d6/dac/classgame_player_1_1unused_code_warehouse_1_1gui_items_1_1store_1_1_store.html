Detailed Description

Store GUI with all the possible towers you can buy. To add a new tower: (1) Create a new Properties File named <TowerName>.properties in the spriteResources.towers package (2) Add the <TowerName> to the Inventory List in the StoreProperties XML file in the gamePlayer.properties.guiItems package (3) Create an Image for the Cell to be displayed in the store with a picture of the tower, its name, and its cost (4) Put the image file in the main voogasalad directory (outside of src)
